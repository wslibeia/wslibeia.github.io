安装GUI图形界面程序

1.在ubuntu上可以这样，通过PPA源安装，仅支持Ubuntu 14.04或更高版本。

sudo add-apt-repository ppa:hzwhuang/ss-qt5 sudo apt-get update sudo apt-get install shadowsocks-qt5

等待安装完成。

2.登录你所购买的ss 更改加密方式和混淆参数为ss/ssr通用 保存 下载gui-config.json 打开shadowsocks软件,导入gui-config.json即可

3.经过上面的配置，你只是启动了sslocal 但是要上网你还需要配置下配置全局代理。 打开google chrome - 设置 - 高级设置 - 打开代理设置 - 网络代理 - 手动 如下图

4.打开google chrome浏览器 输入www.google.hk 即可

关于u盘中.exe病毒问题

gonglja

2017-11-24

safety

1.删除U盘中所有的.exe（可在搜索框中 输入.exe 回车）

2.win + R 打开运行，输入cmd 回车

3.打开我的电脑 找到你U盘的位置 比如说e盘 输入 e: 回车

4.进入u盘后 输入 attrib -h -r -s -a 回车

5.耐心等待一会，你的文件即会恢复
