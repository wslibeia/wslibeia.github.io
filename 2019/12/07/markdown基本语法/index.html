gonglja

2019-12-07

日常

Markdown基础语法，学习必备。

Markdown 语言在 2004 由约翰·格鲁伯（John Gruber）创建。
Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。
Markdown 编写的文档后缀为 .md, .markdown。

2、分割线

在一行中用三个以上的星号、减号、底线来建立一个分割线，也可以在星号或减号中间插入空格，行内不能有其他东西

3、强调

使用星号（*）和底线（_）作为标记强调字词的符号，被 * 或 _ 包围的字词会被转成用 标签包围，用两个 * 或 _ 包起来的话，则会被转成，例如：

single asterisks
single underscores
double asterisks
double underscores

但是如果你的 * 和 _ 两边都有空白的话，它们就只会被当成普通的符号。
