记一次解决双显卡电脑打开软件及系统卡顿的问题

gonglja

2018-06-19

windows

电脑配置

想直接解决问题的，直接跳转到 解决问题

在说这个问题之前，先说下我的电脑配置吧。炫龙KP2（i5-8400 + gtx1060 + 16g +sm961 SSD+ Phison SSD)，sm961（读2000m+,写1000m+）做主硬盘，装系统及常用软件。按道理来说，这配置应该不可能卡顿，但是事实是确实是会卡顿一下，尤其是chrome浏览器（目前就在用这一款浏览器，不确定别的），同学同款电脑同款硬盘也是会卡顿，但是外接显示器不会卡顿，这就使我产生了一个疑问：到底是什么导致的软件启动卡顿？

寻找问题

删注册表

在网上寻找此问题，发现该卡顿类似 “右键卡顿”问题，于是找到右键卡顿的解决方案，一顿乱删，最后只剩两个new 和 workfolders ，打开chrome，发现问题还是没有解决。无奈，只好装系统。（右键卡顿解决方案）

装系统

简单粗暴，在经历简单的备份后，重新装系统（我用的1709）

装好后，在未安装任何软件情况下，安装chrome，启动正常，不卡顿。

安装驱动，当安装独显驱动时，电脑开始卡顿，安装重启完之后，再次打开chrome，启动正常，但是卡顿。

在设备管理器中禁用独显驱动，再次打开chrome，启动正常，没有了之前的卡顿。

以为是系统装的有问题，故又重新装了好几遍，还是同样的问题。

在经过换包（将1709换成1703）之后，重新安装，装完驱动后，发现浏览器卡顿问题减轻，基本上可以忽略，但是打开资源管理器及个性化设置时，还是会卡顿一下。将独显屏蔽后，完全没感觉，所以问题的根源还不是系统，而是独显的原因。

解决问题

首先本文并没有彻底解决问题，想要彻底解决问题，还是等微软老大的更新。

但是有一些很简单的解决办法：

  1. 让N卡的GPU保持活动状态，比如打开N卡控制面板，选到“通过预览调整图像设置”一页，那么N卡的GPU就会一直处于活动状态（任务管理器中可以看出），就不会因为双卡切换延迟造成卡顿。

  2. 禁用核显，只用独显。(bios下禁用，但一般电脑bios下没有该选项)

  3. 不用win10

  4. 换电脑

  5. ……..

每次开电脑都要去打开那个的软件的那个界面，是不是很不方便？

很好，我在搜索该问题时，突然发现了一个Github上jobeid大佬写的解决方案，尝试着试了一下，发现很好用。直接在此链接下下载使用即可。若提示缺少xxx.dll文件，安装Visual C++ Redistributable for Visual Studio 2015 即可。

发现启动电脑后还是需要手动点开那个软件，还是嫌麻烦？好，很好，那我们就让他自己启动。

如何让他自己启动？

首先将你下载 TrayPwrD3.exe 放到任意一个位置，新建快捷方式，然后将快捷方式 移动到

然后你打开任务管理器就会发现已经添加进去了，关机开机，正常运行。到此问题就”解决“了。
